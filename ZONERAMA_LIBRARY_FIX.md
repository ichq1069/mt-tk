## 修复内容

### 1. 转移功能修复

**问题**：
- 转移到壁纸库失败：使用了错误的表名 `media`，实际应为 `media_items`
- 转移到写真库失败：使用了错误的字段名 `image_url`，实际应为 `url`

**修复**：

#### 转移到壁纸库 RPC 函数

**修改前**：
```sql
INSERT INTO public.media (
  user_id,
  url,
  type,
  status,
  title
)
```

**修改后**：
```sql
INSERT INTO public.media_items (
  user_id,
  url,
  type,
  status,
  title
)
```

#### 转移到写真库 RPC 函数

**修改前**：
```sql
INSERT INTO public.album_photos (
  album_id,
  image_url,  -- ❌ 错误字段名
  level,
  sort_order
)
```

**修改后**：
```sql
INSERT INTO public.album_photos (
  album_id,
  url,  -- ✅ 正确字段名
  level,
  sort_order
)
```

**验证**：
- ✅ 转移到壁纸库成功
- ✅ 转移到写真库成功
- ✅ 状态更新正确
- ✅ 数据插入正确

## 新增功能

### 2. 翻页功能

**需求**：
- 图片数量较多时，需要分页显示
- 提升加载速度和用户体验

**实现**：

#### 前端状态管理

```typescript
// 翻页状态
const [currentPage, setCurrentPage] = useState(1);
const [pageSize] = useState(24); // 每页显示 24 张图片（4x6 网格）
const [totalCount, setTotalCount] = useState(0);
```

#### 数据加载逻辑

```typescript
// 获取总数
const { count, error: countError } = await supabase
  .from('zonerama_library')
  .select('*', { count: 'exact', head: true });

setTotalCount(count || 0);

// 获取当前页数据
const from = (currentPage - 1) * pageSize;
const to = from + pageSize - 1;

const { data, error } = await supabase
  .from('zonerama_library')
  .select('*')
  .order('created_at', { ascending: false })
  .range(from, to);
```

#### 翻页控件

**UI 组件**：
- 上一页按钮
- 页码按钮（最多显示 5 个）
- 下一页按钮
- 页码信息提示

**页码显示逻辑**：
```typescript
// 总页数 <= 5：显示所有页码
// 当前页 <= 3：显示 1-5
// 当前页 >= 总页数-2：显示最后 5 页
// 其他情况：显示当前页前后各 2 页
```

**示例**：
```
总页数 = 10，当前页 = 1
显示：[1] 2 3 4 5

总页数 = 10，当前页 = 5
显示：3 4 [5] 6 7

总页数 = 10，当前页 = 10
显示：6 7 8 9 [10]
```

#### 交互优化

**切换页面时**：
- 自动清空选择
- 滚动到顶部（浏览器默认行为）

**获取图片后**：
- 自动重置到第一页
- 刷新图片列表

**页码按钮状态**：
- 当前页：蓝色背景
- 其他页：白色背景
- 上一页（第一页时）：禁用
- 下一页（最后一页时）：禁用

## 技术细节

### 数据库查询优化

**使用 range 查询**：
```typescript
.range(from, to)
```

**优势**：
- 只查询当前页数据
- 减少网络传输
- 提升加载速度

**使用 count 查询**：
```typescript
.select('*', { count: 'exact', head: true })
```

**优势**：
- 只获取总数，不返回数据
- 减少网络传输
- 提升查询速度

### 前端性能优化

**依赖管理**：
```typescript
useEffect(() => {
  loadPhotos();
  // eslint-disable-next-line react-hooks/exhaustive-deps
}, [currentPage]); // 只依赖 currentPage

useEffect(() => {
  loadAlbums();
}, []); // 只加载一次
```

**状态管理**：
- 页码变化时重新加载图片
- 图集列表只加载一次
- 避免不必要的重新渲染

## 使用说明

### 翻页操作

**上一页**：
- 点击「上一页」按钮
- 或点击当前页码左侧的页码

**下一页**：
- 点击「下一页」按钮
- 或点击当前页码右侧的页码

**跳转到指定页**：
- 点击页码按钮直接跳转

**页码信息**：
- 底部显示：「共 X 张图片，第 Y / Z 页」

### 转移操作

**转移到壁纸库**：
1. 选择要转移的图片
2. 点击「转到壁纸库」按钮
3. 在对话框中点击「确认转移」
4. 等待完成：「转移成功！成功 X 张」
5. 在「壁纸管理」→「壁纸库」中查看

**转移到写真库**：
1. 选择要转移的图片
2. 点击「转到写真库」按钮
3. 在对话框中选择目标图集
4. 点击「确认转移」
5. 等待完成：「转移成功！成功 X 张」
6. 在「写真库」→「写真库管理」中查看
